
# THE STRAT ARCADE

A retro neon-themed training app for "The Strat" trading methodology.

## Deployment

### Option 1: Streamlit Cloud (Recommended)
1. Fork this repository to GitHub.
2. Go to [share.streamlit.io](https://share.streamlit.io).
3. Deploy the app from your repo.
4. Set the main file path to `app.py`.

### Option 2: Local Run
1. Install Python 3.11+.
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Run the app:
   ```bash
   streamlit run app.py
   ```

## Features
- **Splash Screen**: Arcade-style start.
- **Tutorial**: Interactive 3-step guide.
- **Game Mode**: Infinite practice with synthetic candle generation.
- **Retro UI**: Neon colors, CRT effects, 8-bit aesthetic.
